== Description ==

This plugin will make possible to interact with your application on Mercado Libre and your Wordpress site.
 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-mercadolibre-sync` to the `/wp-content/plugins/` directory or choose it from plugin repositories.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow instructions on the `Mercado Libre Sync` admin section page.
